* Open **_files -> examples -> badger -> LoRa_temp_hum_**

###### 2. LoRa sensor setup :

* Replace example **_devEUI_**, **_appEUI_** and **_appKey_** with the ones that Your LoRaWANâ„¢ solution provider has assigned.
